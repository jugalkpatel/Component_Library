# Component_Library

Component Library for personal projects built with html, css and vanilla-js.

## Features

- Contains various ready to use components
- Fully responsive

## Components

- Avatar
- Alert
- Badge
- Buttons
- Cards
- Image
- Input
- Lists
- Navigation
- Modal
- Rating
- Slider
- Simplified Grid
- Text Utilities
- Snackbar
